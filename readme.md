# Travel & Go - AI Travel Route Planner

[Български език по-долу](#български)

## English

### Project Overview
**Travel & Go** is a mobile-web application designed to create personalized travel itineraries using Artificial Intelligence. By taking user inputs such as city, budget, time constraints, and travel persona, the app generates optimized routes with specific stops, estimated costs, and real-time weather integration.

This project is an **MVP (Minimum Viable Product)** developed as part of a university project. It serves as a **Proof of Concept** to demonstrate the integration of various APIs and AI models to simplify travel planning.

### Key Features
* **AI-Powered Itineraries**: Generates 3-4 logical route options using LLMs (Llama-3.3 via Groq).
* **Persona Customization**: Tailors routes for Tourists, Food Lovers, or Business Travelers.
* **Real-time Weather**: Fetches current weather data to inform travel suggestions.
* **Dynamic Visuals**: Automatically finds relevant city/location images via Pixabay.
* **Interactive Maps**: Generates deep links to Google Maps with all itinerary waypoints pre-loaded.
* **City Facts**: Provides culturally meaningful and fun facts about the searched destination.

### Technical Stack
* **Frontend**: React, TypeScript, CSS3 (Custom Carousel & Responsive Design).
* **Backend**: Node.js, Express, TypeScript.
* **APIs Used**:
    * Groq (AI Generation)
    * OpenWeatherMap (Weather Data)
    * Nominatim (Geocoding)
    * Pixabay (Image Discovery)
    * Google Maps (Navigation Links)

### Disclaimer
This is a student project. It is intended for demonstration purposes.

## Български

### Общ преглед на проекта
**Travel & Go** е мобилно уеб приложение, създадено за генериране на персонализирани туристически маршрути с помощта на изкуствен интелект. Чрез въвеждане на град, бюджет, времеви рамки и тип на пътешественика, приложението създава оптимизирани маршрути със конкретни спирки, прогнозни разходи и интеграция на метеорологичните условия в реално време.

Този проект е **MVP (Минимално ранен продукт)**, разработен като част от университетски проект. Той служи като **Proof of Concept (Доказателство за концепция)**, за да демонстрира интеграцията на различни API интерфейси и AI модели за улесняване на планирането на пътувания.

### Основни функции
* **Маршрути, генерирани от AI**: Създава 3-4 логически опции за маршрут чрез големи езикови модели (Llama-3.3 чрез Groq).
* **Персонализация**: Адаптира маршрутите за туристи, любители на храната или бизнес пътешественици.
* **Времето в реално време**: Извлича актуални данни за времето, за да предложи подходящи активности.
* **Динамични изображения**: Автоматично намира подходящи снимки на градове и обекти чрез Pixabay.
* **Интерактивни карти**: Генерира директни връзки към Google Maps с всички заредени междинни точки от маршрута.
* **Интересни факти**: Предоставя културни и забавни факти за търсената дестинация.

### Технически стек
* **Фронтенд**: React, TypeScript, CSS3.
* **Бекенд**: Node.js, Express, TypeScript.
* **Използвани API-та**:
    * Groq (AI генерация)
    * OpenWeatherMap (Метеорологични данни)
    * Nominatim (Геокодиране)
    * Pixabay (Изображения)
    * Google Maps (Навигационни линкове)
